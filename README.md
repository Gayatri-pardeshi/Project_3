IntelliMate is a **simple Python desktop AI assistant**. It accepts text commands from the user and gives responses in **text and voice**.

---

## Project Description

This project demonstrates the use of Python for building a basic AI assistant with GUI. It can perform tasks like telling time, weather updates, jokes, and simple web searches.

---

## Features

* Text-based user input
* Voice output using Text-to-Speech
* Graphical User Interface (GUI)
* Current time display
* Weather information
* Random jokes
* Web search

---

## Technologies Used

* Python 3.9+
* Tkinter (GUI)
* gTTS (Text to Speech)
* pygame (Audio playback)
* requests (API calls)

---

## How to Run the Project

1. Install Python 3.9 or above
2. Install required libraries:

   ```
   pip install pygame gTTS requests pytz
   ```
3. Run the program:

   ```
   python Intellimate.py
   ```
